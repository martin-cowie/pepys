# pepys
An ONVIF server for the Raspberry Pi Camera. Pronounced 'peeps'.

## Requirements

### Test Camera Adapter

* The Live555rtsp server.

To install on a Mac `brew install openrtsp`

To install on Linux `sudo apt-get install livemedia-utils`
